### 📚 手动配置

1. 从[官方网站](https://v2.powernukkitx.com/)下载 PowerNukkitX 的最新版本。
2. 将下载的文件解压缩到您的计算机上的一个文件夹中。
<Tabs>
    <TabItem value="Windows">
        3. 打开您解压缩文件的文件夹。
        4. 双击运行 `start.cmd` 文件。
    </TabItem>
    <TabItem value="Linux">
        3. 打开您解压缩文件的文件夹。
        4. 确保 `start.sh` 文件被授予执行权限。
        ```sh
        chmod +x start.sh
        ```
        5. 运行 `start.sh` 文件。
        ```sh
        ./start.sh
        ```
    </TabItem>
</Tabs>
