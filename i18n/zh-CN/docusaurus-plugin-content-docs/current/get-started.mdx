# ⚒️ 快速入门

创建 PowerNukkitX 服务器有两种选择：
- [自动配置](getting-started/automatic/create-server-automatic.md)
> 这个选项通过自动化大部分创建和配置服务器所需的步骤，大大简化了过程。适合寻求快速无忧解决方案的用户。
- [手动配置](getting-started/manual/create-server-manually.mdx)
> 虽然这种方法需要更多的努力和技术知识，但它可以完全控制服务器的配置。这是有经验的用户的首选，他们希望自定义服务器。
