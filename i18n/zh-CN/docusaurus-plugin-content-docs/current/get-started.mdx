# 启动PNX
## 📚通过CLI方式
1. 确保你已经安装了 `JDK21`,如果没有安装，从这里下载[Graalvm JDK21](https://www.graalvm.org/downloads)
2. 下载[`PowerNukkitX-Run.zip`](https://github.com/PowerNukkitX/PowerNukkitX/releases/download/nightly-build/powernukkitx-run.zip)
3. 解压 `PowerNukkitX-Run.zip` 到你想要的目录.
4. 将之前下载好的JDK设置到系统环境变量中, 或解压至与上一步操作相同的目录下.
```sh
java -jar cli.jar
```

## 📚 通过脚本方式
import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

:::tip[]
以下指南需要你通过手动执行启动.如果你使用翼龙面板,请寻找相关Pterodactyl egg **[Pterodactyl](/pterodactyl)**.
:::

1. Download the latest version of PowerNukkitX from the [official website](https://v2.powernukkitx.com/).
2. Extract the downloaded file to a folder on your computer.
<Tabs>
    <TabItem value="Windows">
        3.Open the folder where you extracted the files.
        4.Download the `start.cmd` file from [here](https://raw.githubusercontent.com/PowerNukkitX/scripts/master/start.cmd) and place it in the same folder.
        5.Run the `start.cmd` file by double-clicking on it.
    </TabItem>
    <TabItem value="Linux">
        3.Open the folder where you extracted the files.
        4.Download the `start.sh` file and place it in the same folder.
        ```bash
        wget https://raw.githubusercontent.com/PowerNukkitX/scripts/master/start.sh
        ```
        5.The `start.sh` file needs to be given permissions to run.
        ```bash
        chmod +x start.sh
        ```
        6. Run the `start.sh` file.
        ```bash
        ./start.sh
        ```
    </TabItem>
</Tabs>
