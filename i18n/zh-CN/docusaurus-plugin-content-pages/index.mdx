# 📕 **PowerNukkitX** 的故事

**PowerNukkitX** 的诞生源于为 **Minecraft: Bedrock Edition** 提供更加可定制和强大的服务器软件的愿望。我们的团队由一群对 **Minecraft** 充满热情的开发者组成，他们看到了一个提供更多功能的服务器软件的潜力。

> 我们于 2020 年开始了开发，旨在创建一个不仅强大且可定制，而且友好且适用于所有人的服务器软件。我们想要创建一个让服务器所有者真正定制他们的 **Minecraft** 世界的工具。

2023 年，该项目由 **[CoolLoong](https://github.com/CoolLoong)** 接手，他带来了新的视角和新的想法。在 CoolLoong 的领导下，**PowerNukkitX** 继续增长和发展，同时保持致力于为 **Minecraft: Bedrock Edition** 提供强大且可定制的服务器软件的承诺。

# 🐲 为什么选择 PowerNukkitX？

- **性能**<br/> **PowerNukkitX 专为高性能而优化**，确保即使在玩家众多且世界复杂的情况下，游戏仍能流畅运行。

- **可扩展性**<br/> 它提供了一个**强大的插件系统**，让您可以轻松添加新功能并根据特定需求自定义您的服务器。

- **易用性**<br/> 有了完整的文档和活跃的社区，**PowerNukkitX 用户友好**，既适用于初学者，也适用于经验丰富的服务器管理员。

- **开源**<br/> 作为开源项目，**您可以检查、修改和贡献代码**，促进透明度和创新。

- **定期更新**<br/> 开发团队致力于提供**定期更新**，确保与最新版本的 Minecraft 兼容，并持续添加新功能和改进。

- **社区支持**<br/> 选择 **PowerNukkitX**，您将加入一个**充满活力的社区**，服务器所有者和开发者可以提供支持、共享插件并在项目上合作。

- **独立性**<br/> 作为独立项目，**PowerNukkitX 与 Mojang AB 无关**，为您提供了比官方服务器软件更大的灵活性和创造性自由。

简而言之，**PowerNukkitX 是一个强大且灵活的选择**，适用于任何希望管理高质量 Minecraft Bedrock Edition 服务器的人。

# 🎯 前提条件

在下载 **PowerNukkitX** 之前，请确保您的计算机符合以下要求：

* Windows 10/11、Windows Server、Linux
* Java 21 版本 - [下载 Java 21](https://sdkman.io/jdks)
* 稳定的互联网连接，最好是有线连接
* 能够进行端口转发的调制解调器/路由器（用于在线多人游戏）

:::danger[连接问题]
对于一些人来说，访问他们的服务器或连接到他们的本地服务器几乎是不可能的。在 Windows 终端中以管理员身份运行一个命令可以解决这个问题 [查看命令](tutorials/fix-connection-issues)。
:::

很抱歉，目前我们没有 Debian 和 Linux 的性能数据。这是因为我们的团队仍在收集和分析这些数据。我们理解这些信息对我们的用户很重要，我们正在努力尽快提供。感谢您的耐心等待和理解。
