## Getting Started

- Node.js Version >= 18.x
- PNPM

```bash
npm install pnpm -g
pnpm install
```

```bash
pnpm run start
```

### Add support Language

See: https://docusaurus.io/zh-CN/docs/i18n/git

Use `docusaurus write-translations` to generate the translation files.

```bash
pnpm run write-translations --locale [language]
```
