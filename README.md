## Local Development
To run a local development server, you first need to install the dependencies:

- Node.js Version >= 18.x
- [pnpm](https://pnpm.io/installation)

```bash
pnpm install
```

```bash
pnpm run start
#preview other language
pnpm run start --locale zh-CN
```
