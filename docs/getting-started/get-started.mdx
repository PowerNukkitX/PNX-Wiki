# ⚒️ Quick Start

There are two options to create a PowerNukkitX server:
- Using the Command Line Interface (CLI)
- [Manual Setup](#manual-setup)

This guide will show you how to use the built-in installation program (CLI).

## 📥 Download PowerNukkitX

:::tip
PowerNukkitX requires **Java 21** to run. If you don't have it installed, you can download it from [here](https://www.oracle.com/java/technologies/downloads/#jdk21).
:::

Visit [PowerNukkitX's website](https://v2.powernukkitx.com/) and download the CLI for your operating system.

### 🛠️ CLI Installation

```java
java -jar cli.jar
```

### 📚 Manual Setup

1. Download the latest version of PowerNukkitX from the [official website](https://v2.powernukkitx.com/).
2. Extract the downloaded file to a folder on your computer.
<Tabs>
    <TabItem value="Windows">
        3. Open the folder where you extracted the files.
        4. Run the `start.bat` file by double-clicking on it.
    </TabItem>
    <TabItem value="Linux">
        3. Open the folder where you extracted the files.
        4. The `start.sh` file needs to be given permissions to run.
        ```sh
        chmod +x start.sh
        ```
        5. Run the `start.sh` file.
        ```sh
        ./start.sh
        ```
    </TabItem>
</Tabs>
